## 🚀 Technical Highlights

The portfolio is built with a focus on **modern typography**, **responsive design**, and **smooth user experience**, utilizing a clean "Navy & Gold" aesthetic.

* **Frontend:** HTML5, CSS3 (Custom Variables & Grid/Flexbox Layouts)
* **Fonts:** Playfair Display (Serif), DM Sans (Sans-serif), DM Mono (Monospace)
* **Interactivity:** Vanilla JavaScript Intersection Observer API for scroll-reveal animations.
* **Design Philosophy:** Minimalist, high-contrast, and mobile-first.

## 🛠️ Featured Project: Computer Laboratory Management System
A significant portion of this portfolio highlights my **BSc Final Year Project** developed for the Eastern University of Sri Lanka.

* **Objective:** Automating a 75-computer university lab.
* **Core Tech:** PHP, MySQL, Role-Based Access Control (RBAC).
* **Key Modules:** * **Lab Booking:** Real-time scheduling with conflict detection.
    * **Attendance:** Automated tracking and downloadable reporting.
    * **Maintenance:** A ticketing system for hardware/software issues.
* **System Design:** Structured using EERD and UML Class Diagrams.
